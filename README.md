## 🏦 Elite 101 Bank Bot
This is a simple Python chatbot that acts like a digital bank teller. It helps you sign up for an account, log in, and check out different types of bank accounts (like Student or Premier checking).
## ✨ What it does:

* Sign Up: Asks for your basic info and saves a username/password.
* Log In: Checks a text file to make sure you're actually you.
* Browse Accounts: Shows you cool lists of features for different bank accounts (interest rates, fees, etc.).
* Simple Menus: Uses numbers (1-4) to let you move around the app easily.

## 🛠️ How it works:
It’s all built in Python. It uses a file called database.txt to remember your login info, so you don't lose your "account" when you close the program.
## 🚀 How to run it:

   1. Make sure you have Python installed.
   2. Run the script in your terminal or VS Code.
   3. Follow the prompts to register or log in.
   4. Pick a number to see what the bank offers!

## 📂 Files:

* your_script.py — The main code.
* database.txt — Where your usernames and passwords live.
